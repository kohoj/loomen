# Parity Notes

Loomen targets a dense desktop workflow for agent-driven coding:

- local repositories in a left rail
- per-task worktrees as first-class workspaces
- Scratchpad plus chat tabs in the center
- files, diffs, comments, checks, and PR status in the right inspector
- setup/run/terminal controls in the lower panel
- command palette, slash commands, file mentions, model controls, and permission modes

The implementation is source-available application code. It does not include proprietary binaries, binary assets, private schemas, or vendored Claude/Codex executables.
